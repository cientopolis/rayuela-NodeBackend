# Rayuela backend

## Description

Backend para el framework Rayuela

## Installation
Se puede utilizar cualquier version de node LTS, 16,18 o 20.
Es recomendable utilizar node 20, puedes ejecutar `nvm use 20` o bien `nvm use 20.12.1` - la version que estoy utilizando al iniciar el proyecto -
```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
